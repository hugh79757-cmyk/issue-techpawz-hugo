오토메이터로 이미지 자동 webp 변환하기 - 이미지 용량은 블로그 로딩 속도에 직접적인 영향을 미치며, 이는 곧 검색 엔진 최적화(SEO) 성과로 이어집니다. 특히 구글은 페이지 속도를 순위 요소로 반영하기 때문에, 이미지 파일 포맷과 용량 최적화는 더 이상 선택이 아닌 필수가 되었습니다. 오늘은 **Mac 사용자를 위한 오토메이터(Automator)** 를 활용하여 `.png`나 `.jpg` 이미지를 **자동으로 `.webp`로 변환하는 방법**을 소개합니다. 이 과정은 **이미지 용량을 최대 90%까지 줄여주는 효과**가 있어 블로거에게 매우 유용합니다.

Mac에서 `Command + Shift + 4` 등으로 스크린샷을 찍으면 기본 저장 형식은 `.png`입니다. 그러나 `.png` 파일은 용량이 크고, 웹에서는 `.webp` 형식이 훨씬 효율적입니다. 특히 `.webp`는 동일한 품질을 유지하면서도 파일 크기를 획기적으로 줄여 웹 최적화에 큰 장점이 됩니다.

**이미지 파일을 줄이기 위해 매번 이미지 편집 프로그램을 열고 파일을 불러온 뒤 다시 webp로 변환시키는 과정은 번거롭고 시간 소모가 큽니다.** 이런 수작업을 생략하고 자동화할 수 있다면 블로거에게는 큰 도움이 됩니다. 바로 이럴 때 오토메이터를 활용하면, 파일을 특정 폴더에 저장하는 것만으로 webp 변환이 자동으로 이루어져 **작업 효율을 극대화**할 수 있습니다.

**이 글을 모두 읽을 필요는 없습니다. 위의 과정들을 자신이 사용하는 AI툴에 문의를 하면 친절하게 자신에게 맞는 방법을 알려줄 것이기 때문입니다.**

## 오토메이터로 이미지 자동 webp 변환하기

![오토메이터로 이미지 자동 webp 변환하기](https://issue.techpawz.com/wp-content/uploads/2025/05/1-2025-05-23-pm-5.36.32-1024x869.webp)

### 1. 오토메이터 실행 및 폴더 동작 생성

- Launchpad에서 ‘Automator’를 실행합니다.
- 새 문서를 생성할 때 ‘폴더 동작(Folder Action)’을 선택합니다.
- ‘스크린샷’ 폴더 또는 변환할 이미지가 저장되는 폴더를 지정합니다.

### 2. 작업 구성

- ‘파일 및 폴더’ 항목에서 ‘입력된 파일 복사’를 추가한 뒤 저장할 폴더를 지정합니다 (예: ‘WebP 변환 이미지’).
- ‘유틸리티’ 항목에서 ‘셸 스크립트 실행’을 선택하고 아래 내용을 입력합니다:

```
for f in "$@"
do
  cwebp -q 80 "$f" -o "${f%.*}.webp"
done
```

- 해당 명령어를 사용하기 위해서는 **Homebrew와 webp 패키지 설치**가 필요합니다. 아래 내용을 참고하세요.

## Mac에 Homebrew 설치하기

`cwebp` 명령어를 사용하려면 Mac에 **Homebrew(홈브루)** 를 먼저 설치해야 합니다. Homebrew는 macOS의 터미널에서 각종 유틸리티를 쉽게 설치할 수 있는 필수 패키지 관리자입니다.

![오토메이터로 이미지 자동 webp 변환하기](https://issue.techpawz.com/wp-content/uploads/2025/05/1-2.webp)

### 설치 방법:

1. **터미널 실행**
   - `Command + Space`를 눌러 Spotlight에서 `터미널` 검색 후 실행하세요.
2. **Homebrew 설치 명령어 입력**

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

3. **환경 변수 설정 (zsh 기준)**

```
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
```

4. **설치 확인**

```
brew --version
```

버전이 정상적으로 표시되면 설치 완료입니다.

## webp 변환 도구 설치하기

Homebrew가 설치되었다면, 이제 `cwebp`를 포함한 WebP 도구를 설치합니다:

```
brew install webp
```

설치 후 아래 명령어로 확인합니다:

```
cwebp -version
```

이제 오토메이터 스크립트가 정상 작동합니다.

## 한글 파일명 자동 처리도 가능

페이스북이나 일부 SNS에 블로그 링크를 공유할 때, 이미지 파일명이 **한글일 경우 썸네일이 깨지는 문제**가 발생할 수 있습니다. 이런 경우에도 오토메이터를 통해 **파일명을 영문 또는 UUID로 자동 변경**할 수 있습니다.

예시 셸 스크립트:

```
for f in "$@"
do
  dir=$(dirname "$f")
  ext="${f##*.}"
  uuid=$(uuidgen)
  mv "$f" "$dir/$uuid.$ext"
done
```
