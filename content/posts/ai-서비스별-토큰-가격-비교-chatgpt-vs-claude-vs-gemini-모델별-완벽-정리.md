AI API를 선택할 때 가장 중요한 기준 중 하나가 **가격**입니다. 같은 작업이라도 어떤 모델을 쓰느냐에 따라 비용이 10배 이상 차이 날 수 있습니다. 이 글에서는 2026년 1월 기준 OpenAI, Anthropic, Google의 주요 모델별 가격을 상세히 비교합니다. 각 모델의 특징과 용도별 추천까지 정리했으니 API 선택에 참고하세요.

## 챗지피티 - OpenAI ChatGPT 모델별 가격

OpenAI는 GPT 시리즈로 가장 널리 사용되는 AI API입니다. 모델별 가격은 다음과 같습니다(1M 토큰당 USD 기준).**GPT-4o**는 OpenAI의 주력 모델입니다. 입력 $2.50, 출력 $10.00으로 성능과 가격의 균형이 좋습니다. 128K 컨텍스트 윈도우를 지원하며 텍스트, 이미지, 오디오를 모두 처리하는 멀티모달 모델입니다. 복잡한 대화, 문서 분석, 코드 작성에 적합합니다.

**GPT-4o mini**는 경량화 버전으로 입력 $0.15, 출력 $0.60입니다. GPT-4o 대비 약 17분의 1 가격이면서 간단한 작업에서는 충분한 성능을 보여줍니다. 챗봇, 텍스트 분류, 요약 작업에 가성비 좋은 선택입니다.

**GPT-4.5**는 2025년 출시된 최신 모델로 입력 $75.00, 출력 $150.00입니다. 현존 OpenAI 모델 중 가장 높은 추론 성능을 제공하지만 가격이 매우 높아 특수한 연구나 고난도 작업에만 권장됩니다.

**o3-mini**는 추론 특화 모델로 입력 $1.10, 출력 $4.40입니다. 수학 문제, 논리 추론, 코딩 테스트 등 단계별 사고가 필요한 작업에 최적화되어 있습니다.

## 클로드 - Anthropic Claude 모델별 가격

Anthropic의 Claude는 긴 문맥 처리와 안전성에 강점이 있습니다. Claude 4 시리즈 가격은 다음과 같습니다(1M 토큰당 USD 기준). **Claude Opus 4.5**는 최상위 모델로 입력 $15.00, 출력 $75.00입니다. 200K 컨텍스트 윈도우를 지원하며 복잡한 분석, 창의적 글쓰기, 고난도 코딩에 뛰어난 성능을 보입니다. 다만 가격이 높아 비용 민감한 프로젝트에는 부담이 됩니다.

**Claude Sonnet 4**는 균형 잡힌 모델로 입력 $3.00, 출력 $15.00입니다. Opus의 80% 수준 성능을 5분의 1 가격에 제공합니다. 대부분의 비즈니스 용도에 가장 많이 추천되는 모델입니다. 문서 요약, 이메일 작성, 일반적인 코드 리뷰에 적합합니다.

**Claude Haiku 4**는 경량 모델로 입력 $0.25, 출력 $1.25입니다. 빠른 응답 속도와 저렴한 가격이 장점입니다. 고객 응대 챗봇, 실시간 분류, 간단한 질의응답에 최적입니다.

Claude의 특징은 **프롬프트 캐싱** 할인입니다. 캐싱된 토큰은 기본 가격의 10%만 부과되어 반복 작업에서 비용을 크게 줄일 수 있습니다.

## 제미나이 - Google Gemini 모델별 가격

Google의 Gemini는 대용량 컨텍스트와 멀티모달에 강점이 있습니다. 2026년 기준 가격은 다음과 같습니다(1M 토큰당 USD 기준). **Gemini 2.5 Pro**는 최상위 모델로 입력 $1.25, 출력 $10.00입니다. 1M(백만) 토큰 컨텍스트 윈도우를 지원하여 책 한 권 분량의 문서도 한 번에 처리할 수 있습니다. 긴 문서 분석, 대규모 코드베이스 리뷰에 적합합니다. **Gemini 2.5 Flash**는 속도 최적화 모델로 입력 $0.15, 출력 $0.60입니다(128K 이하 기준). 빠른 응답이 필요한 실시간 서비스에 적합합니다. 128K 초과 시 입력 $0.30, 출력 $1.20으로 가격이 올라갑니다.

**Gemini 2.0 Flash**는 구버전 속도 모델로 입력 $0.10, 출력 $0.40입니다. 가장 저렴한 가격대로 단순 분류, 키워드 추출 등 기본 작업에 충분합니다.

**Gemini Flash-Lite**는 초경량 모델로 입력 $0.075, 출력 $0.30입니다. 대량의 단순 작업을 처리할 때 비용 효율이 가장 좋습니다.

## 모델별 용도 추천 정리

**가성비 최고**: Gemini Flash-Lite(입력 $0.075), GPT-4o mini(입력 $0.15), Claude Haiku(입력 $0.25) 순으로 저렴합니다. 단순 분류, 키워드 추출, 간단한 챗봇에 적합합니다.

**균형 잡힌 선택**: GPT-4o(입력 $2.50), Claude Sonnet(입력 $3.00), Gemini 2.5 Pro(입력 $1.25)가 성능과 가격의 균형이 좋습니다. 일반적인 업무 자동화, 콘텐츠 생성, 코드 작성에 추천합니다.

**최고 성능 필요 시**: Claude Opus 4.5(입력 $15), GPT-4.5(입력 $75)가 최상위 성능을 제공합니다. 복잡한 연구, 고난도 추론, 전문 분야 작업에 사용합니다.

**긴 문서 처리**: Gemini 2.5 Pro의 1M 토큰 컨텍스트가 독보적입니다. 책, 논문, 대규모 코드베이스를 한 번에 분석해야 할 때 선택합니다.

## 실제 비용 예시로 비교하기

한글 3,000자 문서를 요약하는 작업을 가정해봅니다. 입력 약 5,000토큰, 출력 약 500토큰이 발생한다고 가정합니다.

GPT-4o 사용 시: 입력 $0.0125 + 출력 $0.005 = 약 $0.0175(약 24원)입니다. Claude Sonnet 사용 시: 입력 $0.015 + 출력 $0.0075 = 약 $0.0225(약 31원)입니다. Gemini 2.5 Flash 사용 시: 입력 $0.00075 + 출력 $0.0003 = 약 $0.001(약 1.4원)입니다.

동일한 작업에서 모델 선택에 따라 **20배 이상** 비용 차이가 발생할 수 있습니다. 작업 복잡도에 맞는 모델 선택이 얼마나 중요한지 알 수 있습니다.

## 입력 토큰이 더 많은 이유

**문서 요약 작업의 경우**, 입력에는 다음이 모두 포함됩니다:

- **시스템 프롬프트**: "당신은 전문 에디터입니다. 다음 문서를 3문장으로 요약하세요..." (약 100~500토큰)
- **원본 문서 전체**: 한글 3,000자 = 약 4,000~6,000토큰
- **추가 지시사항**: 형식, 톤, 길이 제한 등\

문서 요약 지시를 내릴 때, 입력 토큰의 양이 많고 출력 토큰의 양은 줄어듭니다.

## 마무리
