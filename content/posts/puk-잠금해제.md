## PUK 잠금해제

### PUK 코드란 무엇인가요?

PUK는 ‘Personal Unblocking Key’의 약자로, SIM 카드의 보안 잠금이 활성화됐을 때 이를 해제하는 데 사용하는 **고유한 8자리 숫자 코드**입니다. 일반적으로 다음과 같은 상황에서 필요하게 됩니다.

- PIN 번호 3회 이상 틀림 → SIM 카드 자동 잠금
- 이때 PUK 코드를 입력해야만 다시 사용할 수 있음

PUK 코드도 10회 이상 틀리게 되면, SIM 카드는 완전히 잠기며 **재사용 불가능 상태**가 됩니다. 이 경우 유심을 새로 발급받아야 하므로 주의가 필요합니다.

### PUK 코드 확인 방법

#### 유심 패키지에서 확인하기

휴대폰 개통 시 받은 유심 패키지나 카드 뒷면을 확인해 보세요.  
PUK 코드가 별도로 인쇄되어 있거나, 스크래치로 긁어서 확인할 수 있도록 제공됩니다.

#### 통신사 앱 또는 웹사이트에서 확인하기

통신사에 따라 고객센터 앱이나 홈페이지에서 본인의 PUK 코드를 쉽게 조회할 수 있습니다.

| 통신사 | 확인 경로 |
| --- | --- |
| SKT | T world 앱 > 내 정보 > USIM 관리 |
| KT | 마이케이티 앱 > 가입정보 > 유심 관리 |
| LG U+ | U+ 고객센터 앱 > MY > 유심/단말기 정보 |

#### 고객센터 전화 문의

휴대폰에서 **114**를 눌러 통신사 고객센터에 연결하면 본인 확인 후 PUK 코드를 안내받을 수 있습니다.

- SKT: 114 (무료)
- KT: 114 (무료)
- LG U+: 114 (무료)
- 알뜰폰 사용자는 각 MVNO 고객센터에 문의

### PUK 코드 입력 방법

1. 휴대폰 재부팅 후 SIM 카드 잠금 안내가 나타나면
2. **PUK 코드 입력 화면**에서 8자리 코드를 입력
3. 새로운 PIN 번호 4~8자리 설정 → 확인
4. 재확인 입력 후 SIM 카드 사용 재개

정확한 PUK 코드를 입력하면 다시 정상적으로 통화, 문자, 데이터 사용이 가능합니다.

### PUK 코드도 틀렸다면?

만약 PUK 코드도 10회 이상 잘못 입력한 경우, SIM 카드는 **영구 잠김 상태**가 됩니다. 이 경우에는 다음과 같은 조치가 필요합니다.

- 가까운 통신사 대리점 방문
- 신분증 지참 후 유심 교체 신청
- 교체 비용: 평균 7,000~8,000원 내외 (통신사마다 상이)

유심팩키지 카드, 하단에 PIN이 0000, 그리고 PUK번호 8자리 숫자가 적혀있는 것을 알 수 있습니다.

## 마무리하며

PUK 잠금은 누구에게나 발생할 수 있는 상황입니다. 당황하지 말고 유심 패키지를 먼저 확인하고, 앱 또는 고객센터를 통해 빠르게 해결하세요. 만약 코드를 찾을 수 없다면 대리점을 방문해 새 유심으로 교체하는 것이 가장 확실한 방법입니다. SIM 카드 잠금 해제를 위한 올바른 정보와 빠른 대응만으로도 휴대폰 사용의 불편을 최소화할 수 있습니다.
