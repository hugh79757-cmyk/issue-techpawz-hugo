## 모바일·PC 카카오톡 업데이트 방법

### 모바일 버전 업데이트 방법

**버전 확인**: 카카오톡 실행 → 우측 하단 점 세 개 → 설정 → 앱 관리에서 현재 버전 확인합니다.

**자동 업데이트**: 설정에서 ‘업데이트’ 버튼이 활성화되면 바로 눌러 최신 버전으로 설치합니다.

**수동 업데이트**:

**Android(구글 플레이스토어)**: 스토어에서 ‘카카오톡’ 검색 → '업데이트’ 버튼 클릭합니다.

**iOS(App Store)**: App Store에서 ‘카카오톡’ 검색 → '업데이트’ 버튼 클릭합니다.

### PC 버전 업데이트 방법

**자동 업데이트**: 카카오톡 실행 → 좌측 톱니바퀴 클릭 → 설정 → 정보 탭에서 ‘업데이트’ 버튼 활성 시 클릭하면 설치됩니다.

**수동 업데이트**: 카카오톡 공식 홈페이지에서 Windows 또는 Mac용 설치 파일을 다운로드한 후 실행하여 재설치합니다.
