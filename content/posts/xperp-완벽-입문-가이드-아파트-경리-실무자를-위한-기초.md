아파트 관리사무소에 첫 출근한 경리 담당자라면 XPERP라는 생소한 프로그램 앞에서 막막함을 느끼셨을 것입니다. 전표는 어떻게 입력하는지, 관리비 부과는 어떤 순서로 진행하는지, 수납 처리는 어디서 하는지 하나하나 물어보기도 어렵고 매뉴얼만 봐서는 감이 잡히지 않습니다.

XPERP는 이지스엔터프라이즈에서 개발한 공동주택 전용 ERP 시스템으로, 현재 전국 약 2만여 단지에서 사용하고 있습니다. 아파트 시장의 약 80%가 이 프로그램을 사용할 만큼 업계 표준으로 자리 잡았습니다. 따라서 아파트 경리 업무를 하려면 XPERP 사용법을 반드시 익혀야 합니다.

이 글에서는 XPERP 초보 실무자를 위해 프로그램 설치부터 핵심 기능 활용까지 단계별로 상세히 안내합니다. 이 가이드를 따라 하시면 기본적인 업무 처리에 자신감을 가질 수 있을 것입니다.

## XPERP 프로그램 이해하기

**XPERP(XpERP)** 는 인터넷을 통해 아파트, 빌딩, 주상복합, 오피스텔, 지식산업센터 등 집합건물의 관리 업무를 처리하는 ASP 기반 ERP 시스템입니다. 별도의 서버 구축 없이 웹 브라우저나 전용 프로그램을 통해 접속하여 사용하며, 데이터는 중앙 서버에 안전하게 저장됩니다.

일반 기업에서 SAP, 오라클, 더존, 영림원 같은 ERP를 사용하듯이, 공동주택에서는 XPERP를 통해 회계, 관리비 부과, 수납, 검침, 인사급여 등 모든 관리 업무를 통합 처리합니다. 각 모듈 간 데이터가 유기적으로 연계되어 있어 수작업으로 집계하거나 계산할 필요가 없다는 것이 큰 장점입니다.

XPERP의 주요 메뉴는 단지관리, 입주자관리, 검침, 부과, 수납, 회계관리, 인사급여, 민원관리로 구성됩니다. 입주자 정보를 기반으로 검침 데이터가 연동되고, 이를 토대로 관리비가 부과되며, 부과된 금액은 수납 처리 후 회계 전표로 자동 연계됩니다. 이러한 흐름을 이해하면 프로그램 사용이 훨씬 수월해집니다.

## 프로그램 설치와 로그인

XPERP를 사용하려면 먼저 **XpHUB** 프로그램을 설치해야 합니다. XpHUB는 XPERP를 포함한 다양한 부가 서비스를 통합 제공하는 플랫폼입니다. 설치 과정은 간단하며, 이지스엔터프라이즈 공식 홈페이지에서 설치 파일을 다운로드할 수 있습니다.

설치 방법은 다음과 같습니다. 먼저 이지스엔터프라이즈 홈페이지에 접속하여 XpHUB 다운로드 버튼을 클릭합니다. 다운로드 폴더에서 설치 파일을 실행하고 안내에 따라 설치를 완료합니다. 바탕화면에 생성된 XpHUB 아이콘을 더블 클릭하면 로그인 창이 나타납니다. 관리사무소에서 부여받은 아이디와 비밀번호를 입력하면 프로그램에 접속됩니다.

로그인 후에는 XpERP 아이콘을 클릭하여 메인 업무 화면으로 진입합니다. 상단 메뉴에서 단지관리, 입주자, 검침, 부과, 수납, 회계관리 등 각 업무 영역을 선택할 수 있습니다. 처음 접속 시 메뉴가 보이지 않는 경우가 있는데, 이때는 이지스엔터프라이즈 고객센터에 문의하거나 추가 설치 파일을 받아 적용하면 해결됩니다.

## 전표 입력과 회계 처리

아파트 경리 업무의 핵심은 **전표 처리** 입니다. 관리사무소에서 발생하는 모든 수입과 지출은 전표로 기록되어야 하며, 이 데이터가 관리비 부과와 결산의 기초가 됩니다.

전표 입력 경로는 회계관리에서 전표관리로 이동한 후 전표입력(계정식)을 선택하면 됩니다. 신규 버튼을 클릭하고 전표일자, 전표구분(입금, 출금, 대체), 계정과목, 적요, 금액을 순서대로 입력합니다. 대변 적요 입력 시 F7 키를 누르면 차변 적요가 자동 복사되어 편리합니다. 매월 반복되는 패턴의 전표는 전표복사 기능을 활용하면 업무 시간을 크게 단축할 수 있습니다.

계정과목 추가가 필요한 경우 회계관리에서 기초정보로 이동한 후 계정과목코드관리에서 설정합니다. 상위계정을 선택하고 신규 버튼을 클릭한 뒤 계정과목명을 입력합니다. 예를 들어 제예금 하위에 기업은행 계좌를 추가하려면 제예금을 선택한 상태에서 신규를 클릭하고 기업은행으로 이름을 지정하면 됩니다. K-APT 연동을 위해 국토부 회계 계정과도 매칭 작업이 필요합니다.

## 관리비 부과 프로세스

관리비 부과는 XPERP 업무 중 가장 중요하고 복잡한 영역입니다. 전체 흐름을 이해하면 실수를 줄일 수 있습니다. 부과 프로세스는 부과기초작업, 회계금액집계, 단가처리, 부과금액확인, 부과마감, 출력의뢰 순서로 진행됩니다.

먼저 부과에서 부과처리로 이동한 후 부과기초작업을 선택합니다. 행추가를 클릭하여 산출기간과 납부마감일을 확인하고 저장합니다. 다음으로 관리비부과처리 메뉴에서 회계금액집계 버튼을 클릭합니다. 이렇게 하면 회계에서 발생한 비용 금액이 자동으로 가져와집니다. 자동 집계되지 않는 항목은 당월발생금액란에 직접 입력하고 반드시 저장 버튼을 눌러야 합니다.

금액 입력이 완료되면 단가처리 버튼을 클릭합니다. 면적단가 항목의 경우 발생금액을 입력하면 세대별 단가가 자동 계산됩니다. 부과금액을 확인한 후 수정이 필요하면 재부과를 체크하고 다시 단가처리를 실행합니다. 최종 확인 후 부과마감 메뉴에서 부과마감등록을 진행하고 출력의뢰를 클릭하면 고지서 인쇄가 준비됩니다.

## 검침 관리와 수납 처리

**검침 관리** 는 수도, 전기, 가스, 난방 등 계량기 사용량을 입력하는 업무입니다. 검침 데이터는 관리비 부과에 직접 연동되므로 정확한 입력이 중요합니다.

수도 검침 입력 경로는 검침에서 수도검침으로 이동한 후 월분수신을 선택합니다. 전월지침수신을 클릭하면 이전 달 검침값이 자동으로 불러와지고, 당월 검침값을 입력한 뒤 엔터 키를 누르면 요금이 자동 계산되며 다음 세대로 이동합니다. 동별 수도충당금 조정이 필요한 경우 일괄계산 메뉴에서 적용 동별로 증감률을 입력하여 요금을 조정할 수 있습니다. 검침 자료를 엑셀 파일로 일괄 업로드하는 것도 가능합니다.

**수납 처리** 는 입주민이 납부한 관리비를 처리하는 업무입니다. 수납에서 수납처리로 이동한 후 수납처리를 선택합니다. 수납일과 수납기관을 입력하고 동호수를 검색하여 해당 세대의 납부 금액을 처리합니다. 관리비 통장 입금 내역을 확인하면서 금액과 동호수를 매칭합니다. 입금액으로 수납자를 찾으려면 조건에서 금액을 선택하고 금액 범위를 입력하여 조회하면 됩니다.

수납 완료 후에는 수납전표연계 메뉴에서 회계 전표로 자동 연동 처리합니다. 수납일자를 입력하고 연계 버튼을 클릭하면 수납 금액이 회계 전표로 자동 생성됩니다.

## 자주 사용하는 기능 정리

실무에서 자주 사용하는 기능들을 정리하면 다음과 같습니다.

**중간정산** 은 세입자 전출 시 사용합니다. 입주자에서 전출처리로 이동한 후 동호수를 입력하고 조회합니다. 중간정산 버튼을 클릭하여 정산기간을 수정하고 수도 검침값을 입력합니다. 계산방법을 3개월 평균으로 선택한 뒤 중간정산을 클릭하고 저장 후 영수증을 출력합니다.

**자동이체 세대 등록** 은 부과에서 기초정보로 이동한 후 자동이체세대등록을 선택합니다. 본점중복확인 기능을 통해 이중 신청 세대를 확인하고, 사용하지 않는 은행은 반드시 해지해야 이중 출금을 방지할 수 있습니다.

**고지서 재출력** 은 부과에서 관리비부과조회로 이동한 후 고지서 메뉴를 선택하거나, XPERP 상단의 고지서 메뉴를 직접 클릭하여 진행합니다.

**K-APT 전송** 은 회계관리에서 기초정보로 이동한 후 회계데이터전송을 선택합니다. 회계년월을 부과년월과 동일하게 설정하고 전송 버튼을 클릭합니다.

## 학습 자료와 교육 과정 안내

XPERP 사용법을 더 깊이 배우고 싶다면 다양한 학습 자료를 활용할 수 있습니다. 한국전산기술 홈페이지에서는 XPERP 전체 매뉴얼 PDF를 제공하며, 단지관리, 입주자, 검침, 부과, 수납, 회계, 인사급여, 민원관리 등 모듈별 상세 가이드를 다운로드할 수 있습니다.

유튜브에서 XPERP 또는 아파트 경리실무로 검색하면 실무자들이 올린 무료 강의 영상을 다수 찾을 수 있습니다. 특히 판도라경리실무, 토요일오후, 이지집합건물회계컨설팅 등의 채널에서 초보자를 위한 상세한 튜토리얼을 제공합니다.

체계적인 교육을 원한다면 국비지원 과정을 고려해 보시기 바랍니다. 한국직업능력교육원, 그린아트컴퓨터학원 등에서 아파트경리(XP-ERP) 회계자격취득 과정을 운영합니다. 내일배움카드를 활용하면 교육비 부담을 크게 줄일 수 있습니다. 교육 과정에서는 XPERP 프로그램 실습뿐 아니라 공동주택 회계 이론, 관계 법령, FAT, TAT 자격증 준비까지 함께 다룹니다.
