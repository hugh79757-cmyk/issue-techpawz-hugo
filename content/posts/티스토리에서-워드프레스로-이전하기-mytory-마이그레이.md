티스토리에서 워드프레스로 블로그를 이전하는 방법은 여러 가지가 있습니다. 보통 **티스토리 백업 파일을 이용해 임포트**하거나, **스크래퍼 프로그램으로 공개된 글을 크롤링**해 옮기는 방식이 사용됩니다. 하지만 백업 파일을 직접 임포트하는 과정은 꽤 복잡하고 서버 제약이 많습니다. 그래서 이번 글에서는 [Mytory님](https://mytory.net/)이 개발한 ‘티스토리 마이그레이션 전용 플러그인’을 활용해 간단히 이전하는 방법을 소개합니다.

## 티스토리에서 워드프레스로 이전하기

### 📦 기존 방식 vs Mytory 플러그인

| 구분 | 기존 방식 | Mytory 플러그인 |
| --- | --- | --- |
| 설치 난이도 | 복잡 (직접 임포트) | 간단 (플러그인 업로드 후 실행) |
| 백업 파일 처리 | 수동 업로드 | 플러그인에서 ZIP 업로드 지원 |
| 오류 발생 | 서버 환경에 따라 잦음 | 자동 이어받기 기능 제공 |
| URL 유지 | 수동 설정 필요 | 숫자형 URL은 수동 변경 필요 |

Mytory 플러그인은 **티스토리 백업 ZIP 파일을 업로드하기만 하면 자동으로 글·태그·카테고리까지 불러오는 원클릭 도구**입니다.

---

### 🔄 티스토리 → 워드프레스 이전 절차 요약

[![티스토리에서 워드프레스로 이전하기](https://issue.techpawz.com/wp-content/uploads/2025/12/2025120813415-1024x620.webp)](https://mytory.net/)

1️⃣ **티스토리 관리자 페이지** → 관리 → 블로그 → 데이터 관리 → **‘블로그 백업’ 실행**  
2️⃣ **워드프레스 설치** (웹호스팅은 클라우드웨이즈, 카페24, 케미클라우드 등 선택 가능)  
3️⃣ **Mytory 티스토리 임포터 플러그인 설치**  
4️⃣ 워드프레스 관리자 → **도구 → 티스토리 가져오기** 메뉴에서 ZIP 파일 업로드  
5️⃣ 임포트 완료 후 카테고리·태그·이미지 확인

---

### 💡 플러그인 다운로드 및 설치

- 다운로드: 👉 <https://mytory.net/archives/17445>
- 설치: 워드프레스 관리자 → **플러그인 → 새로 추가 → ZIP 업로드 → 활성화**
- 실행: **도구 → 티스토리 가져오기**

※ 글이 많은 경우 ZIP 파일을 100개 단위로 나누어 업로드하면 오류를 줄일 수 있습니다.  
※ 업로드 용량 제한이 있다면 호스팅 환경에서 PHP 업로드 제한을 늘리세요.

---

### ⚙️ 주요 옵션

- **날짜 유지** → 원본 글 게시일 그대로 유지
- **태그·카테고리 가져오기** → 구조 그대로 복사
- **이미지 WebP 변환** → 페이지 로딩 속도 향상
- **썸네일 자동 생성** → 첫 번째 이미지를 포스트 대표 이미지로 지정

단, WebP 변환은 호스팅 환경에 따라 오류가 발생할 수 있으며 이미지가 많은 경우 임포트 속도가 느려질 수 있습니다.
