ChatGPT, Claude, Gemini 같은 AI 서비스를 API로 사용하면 **토큰** 단위로 요금이 부과됩니다. 토큰이 무엇인지 모르면 예상치 못한 비용이 발생할 수 있습니다. 이 글에서는 토큰의 개념부터 계산 방법, 그리고 비용을 줄이는 실전 팁까지 정리합니다. AI API를 처음 사용하거나 비용 관리가 필요한 분들에게 도움이 될 것입니다.

## 토큰이란 무엇인가

토큰은 AI 모델이 텍스트를 처리하는 **기본 단위**입니다. 사람이 읽는 글자나 단어와는 다르게 AI는 텍스트를 토큰이라는 조각으로 나눠서 이해합니다. 영어 기준으로 1개 토큰은 대략 4글자 또는 0.75단어에 해당합니다. 예를 들어 "ChatGPT is great"라는 문장은 약 4개의 토큰으로 분해됩니다. 한글의 경우 한 글자가 1~2개 토큰을 차지하는 경우가 많아서 영어보다 토큰 소모가 더 큽니다.

토큰은 **입력 토큰**과 **출력 토큰**으로 구분됩니다. 입력 토큰은 사용자가 AI에게 보내는 프롬프트이고, 출력 토큰은 AI가 생성하는 답변입니다. 대부분의 API는 출력 토큰 가격이 입력 토큰보다 2~5배 비쌉니다.

## API 요금 구조의 이해

AI API 요금은 **백만 토큰당 달러** 단위로 책정됩니다. 예를 들어 "입력 $3/1M 토큰, 출력 $15/1M 토큰"이라면 입력 100만 토큰에 3달러, 출력 100만 토큰에 15달러가 부과됩니다.

실제 사용량으로 환산하면 다음과 같습니다. 한글 기준 1,000자 문서는 대략 1,500~2,000토큰 정도입니다. 입력 $3/1M 토큰 요금제에서 1,000자 프롬프트를 보내면 약 0.006달러(약 8원)가 소요됩니다.

요금 구조는 모델 성능에 따라 크게 달라집니다. 최상위 모델인 GPT-4.5나 Claude Opus 4.5는 토큰당 가격이 높고, 경량 모델인 GPT-4o mini나 Claude Haiku는 10분의 1 수준으로 저렴합니다. 따라서 용도에 맞는 모델 선택이 비용 관리의 핵심입니다.

## 토큰 계산 방법

토큰 수를 미리 알면 비용을 예측할 수 있습니다. 각 AI 서비스는 공식 토큰 계산기를 제공합니다.

OpenAI는 **Tokenizer** 페이지(platform.openai.com/tokenizer)에서 텍스트를 입력하면 토큰 수를 확인할 수 있습니다. Anthropic Claude는 API 응답에 사용된 토큰 수가 포함되어 있고, Google Gemini도 마찬가지로 API 응답에 토큰 사용량이 표시됩니다.

간단한 추정 공식도 있습니다. 영어는 **글자 수 ÷ 4**, 한글은 **글자 수 × 1.5~2**로 대략적인 토큰 수를 계산할 수 있습니다. 정확한 비용 산정이 필요하면 공식 도구를 사용하는 것이 좋습니다.

## 비용 절감을 위한 기본 전략

가장 효과적인 방법은 **적절한 모델 선택**입니다. 단순한 텍스트 요약이나 분류 작업에 최상위 모델을 쓸 필요가 없습니다. GPT-4o mini나 Claude Haiku 같은 경량 모델도 충분히 좋은 성능을 보여줍니다. 프롬프트 최적화도 중요합니다. 불필요한 설명을 줄이고 핵심만 전달하면 입력 토큰을 아낄 수 있습니다. 시스템 프롬프트에 반복되는 지시사항은 한 번만 넣고, 대화가 길어지면 요약 후 새 세션을 시작하는 것이 좋습니다.

**프롬프트 캐싱**을 활용하면 동일한 시스템 프롬프트를 재사용할 때 비용을 50% 이상 줄일 수 있습니다. Anthropic Claude와 OpenAI 모두 캐싱 기능을 지원합니다.

## 마무리

AI API 비용은 토큰 단위로 계산되며, 모델 선택과 프롬프트 최적화가 비용 관리의 핵심입니다. 다음 글에서는 프롬프트 길이를 줄이면서도 품질을 유지하는 실전 팁을 더 자세히 다룹니다.
